# 设计规范

## 视觉风格
极简几何风格，受 Bauhaus / 瑞士平面设计影响，以基本几何形构建视觉语言。

## 色彩系统
| 角色 | 色值 | 用途 |
|------|------|------|
| 主色（粉） | `#FF43B4` | 页面背景 |
| 强调色（黄） | `#FFF9C7` | 导航栏、边框、Footer、装饰元素 |
| 黑色 | `#000000` | 作品展示区背景、精选卡片背景 |
| 白色 | `#FFFFFF` | 文字、线条、图标 |
| 灰色 | `#D9D9D9` | 图片占位区 |

## 字体层级

### 主字体 Desolator
- 文件：`fonts/Desolator.woff2`（400 常规）、`fonts/Desolator-Light.woff2`（300 细体）
- 回退栈：Courier New → Noto Sans SC → monospace
- 加载策略：`font-display: swap`（先显示回退字体，加载完成后切换）

| 层级 | 字号 | 用途 |
|------|------|------|
| H1 主标题 | 64px | 姓名 "XUZITENG" |
| H2 副标题 | 48px | "I'm" |
| H3 大标题 | 128px | "Awards" 标题 |
| Body L | 32px | Awards 列表项 |
| Body M | 24px | 角色、欢迎语 |
| Body S | 20px | 分类标签、导航 |
| Body XS | 15px | Footer 链接 |

## 间距系统
基于 4px 网格：
- xs: 4px
- sm: 12px (3×4)
- md: 20px (5×4)
- lg: 32px (8×4)
- xl: 60px (15×4)
- 2xl: 80px (20×4)
- 3xl: 120px (30×4)

## 设计基准
- 设计稿宽度：自适应屏幕宽度
- 圆角：仅用于圆形元素（50%），其余直角
- 线条粗细：3px（统一描边）
- 图标风格：线性描边（与装饰图形一致）

## 组件规范

### 作品缩略图卡片
- 外框：228×228px，黄色 `#FFF9C7`
- 内图：196×196px，灰色 `#D9D9D9`，偏移 16px
- 间距：102px

### 精选卡片
- 外框：312×596px，黑色
- 黄色层：272×530px
- 内图：234×492px
